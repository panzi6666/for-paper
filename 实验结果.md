## 实验说明

1. **TCN-PSTA模型**
   - 使用时间卷积网络（TCN）结合PSTA注意力机制
   - 训练轮次：100轮
   - 最终验证F1分数：0.6690

2. **STGCN-Baseline模型**
   - 使用时空图卷积网络（STGCN）作为基准模型
   - 训练轮次：100轮
   - 最终验证F1分数：0.9784

3. **STCBAM-PSTA模型**
   - 使用时空图卷积网络结合CBAM注意力机制和PSTA
   - 训练轮次：100轮
   - 最终验证F1分数：0.8920
   - 最佳验证F1分数：0.9156

4. **STGCN-NoCBAM模型**
   - 使用时空图卷积网络，不包含CBAM注意力机制
   - 训练轮次：100轮
   - 最终验证F1分数：0.7063
   - 最佳验证F1分数：0.7656

5. **STGCN-NoPSTA模型**
   - 使用时空图卷积网络，不包含PSTA注意力机制
   - 训练轮次：100轮
   - 最终验证F1分数：0.9556
   - 最佳验证F1分数：0.9571

## 实验结论

1. **模型性能排名**：
   - STGCN-Baseline > STGCN-NoPSTA > STCBAM-PSTA > STGCN-NoCBAM > TCN-PSTA

2. **注意力机制的影响**：
   - CBAM注意力机制对模型性能有显著影响，移除后性能下降明显
   - PSTA注意力机制的影响相对较小，移除后模型仍能保持较高性能
   - 两种注意力机制结合（STCBAM-PSTA）的效果不如单独使用

3. **模型架构的影响**：
   - STGCN架构整体优于TCN架构
   - 基础STGCN模型（Baseline）表现最好，说明模型架构本身已经足够强大

4. **过拟合分析**：
   - 所有模型都存在一定程度的过拟合
   - STGCN-Baseline和STGCN-NoPSTA的泛化能力最好
   - TCN-PSTA的过拟合问题最严重
